# 주요 기능

\[재고 관리]

* 재고 조회
* 출고, 수량 변경, 위치 변경
* 출고 품목 출력
* 재고 등록

\[부품 관리]

* 부품 조회
* 부품 등록 및 삭제

\[위치 관리]

* 위치 조회
* 위치 등록 및 삭제

\[전체 데이터 백업 및 복구]
